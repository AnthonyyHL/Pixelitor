Pixelitor Installation

Starting with the version 4.3.0 Pixelitor requires Java 17 or higher. The Windows installer contains the required Java, but the other downloads don't contain the Java Runtime Environment.
  • Windows:
    • You can just run the installer progeam, it will install everything. If you prefer to use the jar file, you need to also download and install the latest Java OpenJDK from here. When installing OpenJDK, make sure to select the "Set JAVA_HOME variable" option.
    • Pixelitor:
      • Just run the setup program. It is not necessary to uninstall first a previous version.
      • Alternatively you can also download the jar file and double-click on that to start the program. (Less user-friendly, but it does not require administrator privileges, and it can be used as a "portable application")

  • Linux: you can install Pixelitor either using Flathub or manually. Please note that Pixelitor can't currently use ImageMagick if you are using the Flathub installation.
    • Flathub: see the Pixelitor page on Flathub
    • Manually:
      • Java: You can use the newest OpenJDK package for your Linux distribution.
      • Pixelitor: Run it as a Java program according to your distribution guidelines (double-clicking on the downloaded jar file should work, the command line java -jar pixelitor_VERSION.jar always works).

  • Mac:
    • Java: You might need to install the newest Java from here.
    • Pixelitor: Double-click on the downloaded jar file to start the program.
